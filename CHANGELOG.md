# Changelog

## 0.3.6

- Completed the public plugin shell and metadata for `OpenCalendarKit`.

- Added automated unit, harness-integration, and WordPress-core-near test layers.
- Prepared GitHub, WordPress.org-style, and packaging documentation for distribution.
